# ✊🤚✌️ Rock, Paper, Scissors

> An interactive, web-based version of the classic game built as a personal project to practice core JavaScript logic and DOM manipulation.

## 📝 Overview

This is a fast-paced Player vs. Computer game of Rock, Paper, Scissors. The user selects their move, and the system instantly generates a randomized counter-move. The app evaluates the winner based on classic game rules and updates the session scoreboard in real-time.

## ✨ Features

- **Player vs. Computer:** Play against a randomized system opponent.
- **Dynamic Scoreboard:** Keeps a running tally of Player Wins, Computer Wins, and Draws during the active session.
- **Instant Feedback:** The UI immediately updates to show what the computer chose and who won the round.
- **Responsive Design:** Styled cleanly with CSS to work smoothly on both desktop and mobile screens.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and button inputs.
- **CSS3:** Flexbox, interactive hover states, and responsive styling.
- **Vanilla JavaScript (ES6+):**
  - `Math.random()` for the computer's choice generation.
  - Event Listeners for user input.
  - Conditional logic (`if/else`) for win/loss/draw evaluation.
  - DOM manipulation for live UI and score updates.
